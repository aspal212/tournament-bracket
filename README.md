# Tournament Hub
Website statis untuk mengelola pertandingan sistem gugur, gugur ganda, setengah kompetisi, dan kompetisi penuh.  
Dibuat agar bisa langsung di-hosting di GitHub Pages tanpa build step.

## Cara pakai
1. Fork repo ini
2. Aktifkan GitHub Pages: Settings → Pages → Branch: main → Save
3. Akses `https://username.github.io/tournament-hub`
4. Pilih jenis turnamen dan masukkan data peserta
